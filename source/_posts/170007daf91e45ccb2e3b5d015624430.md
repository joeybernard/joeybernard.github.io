I have been an Obsidian user for a very long time. And I have loved it. But, as I have wanted to do more and more with it, I have bumped up into more and more pain points. I am the first to point out that these pain points are because it is not fitting the way I want to work. I want to be able to feed these notes into other processes and workflows. While there are some plugins available to do this within Obsidian, it doesn't feel as capable as the plugins available within Joplin. For example, I am using a Joplin plugin to take this blog posting and pushing it up to github to be available on this blog. There are also plugins to manage cross-posting to other sites. I think that I will be moving all of my current notes over to Joplin, at least as an experiment.
